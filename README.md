## How to Run Project Locally

Clone the project

```bash
  git clone https://github.com/elhaidar/tes-muatmuat.git
```

Go to the project directory

```bash
  cd tes-muatmuat
```

Install dependencies

```bash
  npm install
```

Start the server (development)

```bash
  npm run dev
```
